### Ден 1: Основи на Gymnasium + DQN демо

1. **Инсталирайте библиотеките:**
   ```bash
   pip install gymnasium[classic-control,box2d] stable-baselines3[extra] tensorboard
   ```

2. **Стартирайте случаен агент върху CartPole:**
   ```python
   import gymnasium as gym
   env = gym.make("CartPole-v1", render_mode="human")
   obs, info = env.reset()
   for _ in range(1000):
       action = env.action_space.sample()  # случайно действие
       obs, reward, terminated, truncated, info = env.step(action)
       if terminated or truncated:
           obs, info = env.reset()
   env.close()
   ```
   *Наблюдение: количката пада почти веднага. Това е вашата базова линия — случайна стратегия.*

3. **Тренирайте DQN върху CartPole със SB3:**
   ```python
   from stable_baselines3 import DQN
   model = DQN("MlpPolicy", "CartPole-v1", verbose=1, tensorboard_log="./logs/")
   model.learn(total_timesteps=50_000)
   model.save("dqn_cartpole")
   ```
   *Наблюдение: дневникът на обучението показва как наградата се изкачва от ~10 до 500 (максимум). Агентът СЕ НАУЧИ.*

4. **Оценете тренирания агент:**
   ```python
   from stable_baselines3 import DQN
   from stable_baselines3.common.evaluation import evaluate_policy
   import gymnasium as gym
   model = DQN.load("dqn_cartpole")
   env = gym.make("CartPole-v1")
   mean_reward, std_reward = evaluate_policy(model, env, n_eval_episodes=10)
   print(f"Mean reward: {mean_reward:.2f} +/- {std_reward:.2f}")
   ```

5. **Визуализирайте с TensorBoard:**
   ```bash
   tensorboard --logdir ./logs/
   ```

### Ден 2: PPO върху LunarLander + Проучване на хиперпараметри<sup class="gl" data-gl="hyperparameter">gl</sup>

1. **Тренирайте PPO върху LunarLander:**
   ```python
   from stable_baselines3 import PPO
   model = PPO("MlpPolicy", "LunarLander-v3", verbose=1, tensorboard_log="./logs/")
   model.learn(total_timesteps=200_000)
   model.save("ppo_lunarlander")
   ```

2. **Експериментирайте с хиперпараметри** — Променяйте по ЕДИН параметър наведнъж, тренирайте отново и сравнявайте кривите в TensorBoard:
   - Скорост на обучение (Learning rate): опитайте `1e-4`, `3e-4` (по подразбиране), `1e-3`
   - Размер на мрежата: опитайте `policy_kwargs=dict(net_arch=[64, 64])` срещу `[256, 256]`
   - Дисконтиращ множител (gamma): опитайте `0.99` (по подразбиране), `0.95`, `0.999`
   - Диапазон на изрязване (Clip range): опитайте `0.1`, `0.2` (по подразбиране), `0.3`

3. **Наблюдавайте тренирания агент:**
   ```python
   model = PPO.load("ppo_lunarlander")
   env = gym.make("LunarLander-v3", render_mode="human")
   obs, info = env.reset()
   for _ in range(1000):
       action, _ = model.predict(obs, deterministic=True)
       obs, reward, terminated, truncated, info = env.step(action)
       if terminated or truncated:
           obs, info = env.reset()
   ```

**Връзка:** http://incompleteideas.net/book/the-book-2nd.html (безплатен HTML/PDF)

**Контекст — какво следва (Глави 7–17):** Глави 7–8 обхващат n-стъпкови методи и следи от пригодност (eligibility traces) — подобрения на методите за времеви разлики (Temporal Difference, TD). Глави 9–11 обхващат апроксимация на функции (мостът към Deep RL — ще срещнете това в Стъпки 5–6). Глави 14–16 обхващат връзки с психологията/невронауката. Глава 17 обхваща изследователските граници. *Глави 7–8 са полезен фон, но не са критични за Стъпка 1; апроксимацията на функции в Глави 9–11 е мястото, откъдето идва „дълбокото“ в Deep RL и ще бъде уместно от Стъпка 5 нататък.*  
**Фокус при четене:**
- **Гл. 1–2 (преглед, ~1 ч):** Рамкиране и многоръки бандити (multi-armed bandits). Виждали сте това във видеоклиповете от Фаза 1. Прегледайте за нотацията.
- **Гл. 3 (четете внимателно, ~2 ч):** Формулировка на MDP. Това е формалният език за всичко следващо. Уверете се, че разбирате: състояние (state), действие (action), награда (reward), преход (transition), стратегия (policy), функция на стойността<sup class="gl" data-gl="value_function">gl</sup> (value function), уравнение на Белман.
- **Гл. 4 (прочетете 4.1–4.4, прегледайте останалото, ~2 ч):** Динамично програмиране. Оценка на стратегията, итерация на стратегията, итерация на стойността. Това са решенията, „ако имахте модела“.
- **Гл. 5 (прочетете 5.1–5.5, ~2 ч):** Методи Монте Карло. Обучението без модел започва тук. MC при първо посещение, проучващи начала (exploring starts).
- **Гл. 6 (четете внимателно, ~3 ч):** TD обучение — ядрото на съвременното RL. TD(0), SARSA, Q-обучение (Q-learning). Това е главата, която се свързва с DQN.
- **Гл. 13 (прочетете 13.1–13.4, ~2 ч):** Методи с градиент на стратегията<sup class="gl" data-gl="policy_gradient">gl</sup> — алгоритъм REINFORCE. Това се свързва с PPO.
